B. Jenis-jenis IP Config

  • ipconfig /all: Tampilkan detail lengkap alamat fisik (MAC) dan IP.
  • ipconfig /release: Melepas alamat IP saat ini.
  • ipconfig /renew: Meminta alamat IP baru dari server DHCP.
  • ipconfig /flushdns: Membersihkan cache DNS agar data web terbaru bisa diakses.

C. IP Address

Internet Protocol Address adalah label numerik unik untuk identifikasi perangkat di jaringan.

Kelas A: 10.10.0.2 | Mask: 255.0.0.0 | Gateway: 10.10.0.1
Kelas B: 172.16.0.2 | Mask: 255.255.0.0 | Gateway: 172.16.0.1
Kelas C: 192.168.0.2 | Mask: 255.255.255.0 | Gateway: 192.168.0.1
