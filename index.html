C.ip adres

apa itu ip adres

ip address (Internet Protocol Address) adalah label numerik unik 32-bit (IPv4) yang diberikan ke setiap perangkat yang terhubung ke jaringan internet atau lokal untuk komunikasi

berdasar kan kelas nya ip adres di bagi menjadi 3 kelas yaiutu

kelas A :10.10.0.2
gate way:10.10.0.1
subnet mask:255.0.0.0

kelas B:172.16.0.0
gate way:172.16.0.1
subnet mask:255.255.0

kelas c:192.168.0.1
gate way:192.168.0.1
subnet mask:255.255.255

kelas D:224.0.0.2
gate way:224.0.0.2

G.macam-macam aplikasi virtual

1.virtual box


fungsi & kegunaan nya= di gunakan untuk uji coba aplikasi menjalankan soft ware lama/belajar sistem operasi baru (seperti linux) tanpa merusak komputer lama

adapun cara konfigurasi & manfaat nya=sangat mudah cukup instal file exe/dmg lalu buat vm baru melalui antar muka (GUI) yang ramah pengguna,& manfaat nya gratis lintas platform,mendukung banyak fitur dasar seperti snapshot (menyimpan keadaa VM)

2.vm ware

fungsi dan kegunaan=vm ware adalah peminpin industri virtualisasi mereka memiliki 2 lini utama:work station(type-2 untuk desktop) dan exic (type1) (untuk type -1 server)

konfogurasi dan manfaat=lebih kompleks dan profesional & untuk manfaat nya sangat stabil,peforma tinggi dan memliki manajemen jaringan yang sangat kuat

3.proxmox virtual envirotmen (VE)

fungsi & kegunaan nya mengelola server fisik menjadi klaster server vritual.sangat populer untuk home lab dan penyedia layanan hosting.

manfaat dan konfigurasi=di lakukan memlalui antar muka berbasi web (web GUI) setelah instalasi selesai tidak burtuh apkikasi remote khusus,cukup pakai browser & manfaat nya gratis tanpa batasan fitur,& sangat efisien dalam penggunaan sumber daya
